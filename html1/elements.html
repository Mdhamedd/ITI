HTML Elements

What are HTML Elements?

HTML elements are the building blocks of HTML pages. They are made up of a start tag, content, and an end tag.

<tagname>Content goes here...</tagname>

Start Tag Element Content End Tag
<h1> My First Heading <h1/>
<p> My First Paragraph <p/>
<br> none none


Nested HTML Elements

HTML elements can be nested.
All HTML documents consist of nested HTML elements.
The following is an example:

Example

<!DOCTYPE html>
<html>
<head>
<title> My First Heading </title>
</head>
<body>

<h1> This is a Heading </h1>
<p> This is a paragraph. </p>

</body>
</html>

HTML Tags

HTML tags are used to define elements in an HTML document. Tags are enclosed in angle brackets and come in pairs, with an opening tag and a closing tag.

Common HTML Elements

  • <h1> to <h6> Headings
  • <p> Paragraph
  • <a> Link
  • <img> Image
  • <div> Division
  • <span> Span
