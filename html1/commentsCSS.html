CSS Comments


What Are CSS Comments?

Comments are used to explain the code, and may help when you edit the source code at a later date. Comments are ignored by browsers.
A CSS comment is placed inside the <style> element, and starts with /* and ends with */:

Single-line

/* This is a single line comment */
p{
 color:red
}

Multi-line

/*This is a multiline comment
where none of these lines will be read by the browser
and is all used to help readability*/
p{
 color:red
}
