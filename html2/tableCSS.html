CSS Tables


To specify table borders in CSS, use the border property.
The example below specifies a solid border for (table,th,td) elements:
It also shows how a table can take full width

Name Age
Mr. Jones 50

Example

table, th, td {
  border: solid 2px #04aa6d;
  width: 100%;
}
Try it Yourself »

As you can see in the examples above have double borders, which doesn't look very pretty. To solve this we use the border-collapse property

Example

table, th, td {
  border: solid 2px #04aa6d;
  width: 100%;
  border-collapse: collapse;
}

Now it looks much better!

Table Size

Table Width and Height

The width and height of a table are defined by the width and height properties.
The example below sets the width of the table to 100%, and the height of the elements to 70px:

Company Contact Country
Alpha Doe John Smith USA

Example

table {
  width: 100%;
}
th {
  height: 70px;
}

Table Alignment

Horizontal Alignment

The text-align property sets the horizontal alignment (like left, right, or center) of the content in <th> or <td>.
By default, the content of <th> elements are center-aligned and the content of <td> elements are left-aligned.
To center-align the content of <td> elements as well, use text-align: center:

Company Contact Country
Alpha Doe John Smith USA

td {
  text-align: center;
}

Vertical Alignment

The vertical-align property sets the vertical alignment (like top, bottom, or middle) of the content in <th> or <td>.
By default, the vertical alignment of the content in a table is middle (for both <th> and <td> elements).
The following example sets the vertical text alignment to bottom for <td> elements:

Company Contact Country
Alpha Doe John Smith USA

td {
  height: 50px;
  vertical-align: center;
}

Table Responsive

Responsive Table

A responsive table will display a horizontal scroll bar if the screen is too small to display the full content:

Tip: Minimize the screen to display the scroll feature!

John Smith 53 53 53 53 53 53
Mary Vasques 64 69 63 82 118 53

Add a container element (like <div>) with overflow-x:auto around the element to make it responsive:

Example

<div style="overflow-x:auto;">
<table>

table content....

</table>
</div>
