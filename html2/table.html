HTML Tables

What are HTML Tables?

HTML tables allow developers to arrange data into rows and columns

Define an HTML Table

A table in HTML consists of table cells inside rows and columns.

Example

A simple HTML table:

<table>
  <tr>
   <th>Company</th>
   <th>Contact</th>
   <th>Country</th>
  </tr>

  <tr>
   <td>Mohamed Hamed</td>
   <td>Ahmed Mohamed</td>
   <td>Egypt</td>
  </tr>

  <tr>
   <td>John Doe</td>
   <td>Mary Jones</td>
   <td>Brazil</td>
  </tr>

</table>


Basic Elements of Table
Element Tag Position
Table Rows <tr> Inside <table>
Table Cells <td> Inside <tr>
Table Headers <th> Inside <tr>

By default content inside <th> is bold and centered

Table Borders

HTML tables can have borders of different styles and shapes.

How to add a border

To add a border, use the CSS border property on table, th, and td elements:


Example

table,th,td {
  border: 1px solid black;
}

Round Table Borders

To style borders and make them curved you can use the border-radius CSS property:

Example

table,th,td {
  border: 1px solid black;
  border-radius: 10px;
}

There's many many more properties that can make our tables and borders look much better but that will all be discovered in css. What's mentioned above is enough to get you going in HTML!

Table - Colspan

To make a cell span over multiple columns, use the colspan attribute:


Example

The value of the colspan attribute represents the number of columns to span.
<table>
  <tr>
   <th colspan="2">Name</th>
   <th>Age</th>
   </tr>

  <tr>
   <td>Jill</td>
   <td>Smith</td>
   <td>43</td>
  </tr>

  <tr>
   <td>Eve</td>
   <td>Jackson</td>
   <td>57</td>
  </tr>

</table>
 

Table - Rowspan

To make a cell span over multiple rows, use the rowspan attribute:


Example

The value of the rowspan attribute represents the number of rows to span.
<table>
  <tr>
   <th>Name</th>
   <th>Age</th>
   </tr>

  <tr>
   <td rowspan="2">Phone</td>
   <td>43</td>
  </tr>

  <tr>
   <td>0109826755</td>
   </tr>

</table>
